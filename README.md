# Superstore Sales Dashboard (Excel)

## Project Overview

This project is an interactive Sales Dashboard built in Microsoft Excel to analyze sales performance, profitability, and business trends using Superstore data.

The dashboard enables users to monitor key business metrics, identify top-performing categories, analyze regional performance, and uncover actionable insights through interactive visualizations.

---

## Tools & Technologies

* Microsoft Excel
* Power Query
* Pivot Tables
* Pivot Charts
* KPI Cards
* Slicers

---

## Dashboard Features

* Interactive filtering using slicers
* Automated data transformation with Power Query
* Dynamic KPI tracking
* Sales and profit analysis
* Category and regional performance analysis
* Trend visualization for business decision-making

---

## Key Metrics Analyzed

* Total Sales
* Total Profit
* Profit Margin
* Sales by Category
* Sales by Region
* Top Performing Products
* Sales Trends Over Time
